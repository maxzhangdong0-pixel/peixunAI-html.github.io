DEFINITION & CORE ROLE

指令、文本与约束的底层聚合

提示词（Prompt）是人类向大语言模型（LLM）传达文本指令、上下文环境、特定问题或输出约束的最小输入单元。

单次调用指令
解决多轮/单轮交互中“一句话如何让AI输出达标”的优化。
构成组件化
常包含：角色、任务指令、Few-shot示例、思维链 (CoT)。

SYSTEM COMPONENT PACKAGE

AI Skill 的标准化内部装配体系

Skill 是比 Prompt 更高维度的抽象单元，相当于为大语言模型预装的一套标准化业务操作流程（如“合同审核 Skill”、“周报自动生成 Skill”）。

01 固定提示词链条

Skill 内部包含了完成特定任务所需的高质量多阶提示词和参数注入点。

02 工具调用规则 (Tools)

明确配置大模型应在何时调度网络检索、计算器或数据库检索等外部接口。

03 无需微调模型参数

无需重训模型，完全依靠上下文控制实现轻量化热插拔和可复用机制。

04 规范的入参/出参

设计严密的 JSON 格式约束，让输入输出满足自动化开发集成对接的硬性门槛。

03
AUTONOMOUS LOOP ENGINEERING

循环工程：自主迭代的自治闭环系统

LOOP_CYBERNETIC_MODEL.svg

自治反馈架构：目标 → 执行 → 校验 → 循环 → 退出。完全脱离人类反复手写指令的物理局限。

CYBERNETICS & AUTONOMY

无人值守的自动化运行系统设计

循环工程（Loop Engineering）设计的是一个高度自治、自我纠错、自我迭代的自治闭环机制，它的终极宿命是彻底代替人类手工的一轮轮发问与校对。

自动自治化交互

系统接收全局总目标后，自动执行多次内部交互循环，自动校验，直到输出合格的成果，自主达成退出逻辑。

精密的退出条件 (Stop Criteria)

配备异常容错，当出现无限循环时能优雅重试、终止并向人类报送健康指标日志。

04
SYSTEM DEPENDENCY LAYER

层级递进：Prompt → Skills → Loop Engineering

LAYER 01 - ATOM UNIT

提示词是底层基石

所有的 Skill 内部均需要由多条精心雕琢的提示词承载运行。Loop 的每一次原子轮转执行，其本质也是向大语言模型终端输送 Prompt 载荷，这是模型唯一能听懂的思考原料。

没有提示词，就没有思维引擎
LAYER 02 - CONTAINER UNIT

Skill 提供可调度能力

Skill 是对零散提示词进行的高度内聚封装。在上层自动化逻辑（Loop）运行时，系统不需要笨拙地向模型发出一大组杂乱提示词，而是像乐高积木一样，快速调度一个个开箱即用的标准 Skill。

实现提示词标准化工程复用
LAYER 03 - PILOT UNIT

Loop 是顶层上级调度系统

传统提示词靠人手工循环、靠人脑肉眼校验。而 Loop 则是系统级的闭环控制器。人只用在顶层下达一次宏大任务指令，Loop 负责串联多个 Skill，自主进行多重轮转运行。

实现 AI 深度自治运行
COLLABORATION WORKFLOW

三者全链路闭环协调运行范式

1
开发者预先封装设计

设计完备的结构化提示词（Prompts），把它们整体打包装入标准化工具中，组合为可插拔调用的 Skills（例如“文档格式摘要 Skill”、“系统联试测试 Skill”）。

2
引入 Loop Engineering 控制系统

定义顶层目标、重试上限、退出容灾条件和严密的机器自动检查规则。该系统能够自主轮转流转，完全脱离人工干预。

3
闭环系统无休运行

系统自动运行，调度 Skill 1 并自动通过模型提问，获取回复后如遇格式残损，系统自动组织错误日志通过 Prompt 指令再次发问并校验，直到完美后流转到下一阶段。

05
INTUITION & ANALOGY

通俗类比：做菜步骤、菜谱与全自动烹饪系统

Atomic Action

提示词 = 做菜单个步骤

具体的指令细节：“切好肉丝备用、开大火爆香葱蒜、撒两勺盐调味”。每一条都是具体的、孤立的动作指令，要求精准，不可有歧义。

Standard Recipe

Skills = 鱼香肉丝菜谱

一个标准化、固化成册的操作手册：将切肉、爆炒、勾芡、出锅等繁琐的爆炒提示词彻底打包、标注火候。下厨时只要打开这本固定菜谱即可精准还原出鱼香肉丝的口味。

Autonomous Machine

Loop = 全自动炒菜机器人

一整套全智能机械化厨房：倒入原材料后，自动读取对应菜谱（Skill），自主计算翻炒时间。烹饪完毕自动尝味（机器校验），发现味道淡了自动回锅重炒，直到完美的食物装盘才会优雅关机。
